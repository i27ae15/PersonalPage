Sudoku Solver

This project was created on my own for solving Sudoku puzzles. The project was built using Flask for the backend and involved learning and implementing various algorithms for solving Sudoku puzzles.

The platform provides an easy-to-use interface for inputting a Sudoku puzzle, and then it utilizes several algorithms to solve the puzzle and display the solution.

As the sole developer on the project, I learned and implemented several algorithms for solving Sudoku puzzles.

The Sudoku Solver project demonstrates my skills as a developer and my ability to learn and implement complex algorithms for solving problems.
