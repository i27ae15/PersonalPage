Pantry

This projected was created completely on my own. The project is an administration platform for managing a pantry, and it's built using Python and Flask for the backend, and Flask Jinja and Vanilla JS for the frontend.

As the sole developer on the project, I designed the platform to be user-friendly, intuitive, and efficient. The backend was built with Python and Flask, which allowed me to create robust functionality and implement effective data management. I used Flask Jinja and Vanilla JS for the frontend, which provided a seamless user experience and allowed for the easy management of pantry items and data.

Overall, the administration platform I created is an excellent solution for efficiently managing a pantry. The project demonstrates my skills as a full-stack developer and my ability to create a project from start to finish on my own.
